The features implemented are:
* Contact picker
* Search using flickr api
* Photos loaded asynchronously
* List item is highlighted when it is clicked
* Each list item has date taken, description, owner name
* If the photo is clicked, a browser is invoked with medium sized version of the photo

Features not implemented are:
* Pull up to load more

Libraries used :
* Lazy list - to list the images in listview.Images are being downloaded asynchronously in the background. Images are being cached on SD card and in memory.
